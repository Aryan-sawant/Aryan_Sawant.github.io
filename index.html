RAG system using LangFlow and DataStax VectorDB (Agentic AI)

An end-to-end Agentic AI RAG system using LangFlow and DataStax VectorDB without writing any code. Just drag, drop and configure the components. The system processes data, embeds it using Astra Vectorizers, and retrieves answers via a database agent (Groq API) or a web search agent (Google API).
